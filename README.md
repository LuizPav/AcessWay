# AcessWay
Projeto em Kotlin para a cadeira de Desenvolvimento Móvel. 

# Objetivo
O objetivo é a construção de uma aplicação com integração a uma API para resgatar paradas de ônibus pŕóximas e por meio de um sistema de avaliação determinr se essa parada está em boas condições (acessivel ou não) visto que o publico alvo são epssoas com deficiência
